## üìä Using PyDebFlow
Once you have installed PyDebFlow, you can launch the application. Follow these steps to start simulating:

### Step 1: Open the Application
- Locate the PyDebFlow application icon on your desktop or in your applications folder.
- Double-click to open.

### Step 2: Load a DEM File
- Click on the 'Load DEM' button to select a Digital Elevation Model (DEM) file. This is a terrain model that represents the surface.
- Supported formats include TIFF.

### Step 3: Configure Simulation Parameters
- Choose physics-based models from the options provided.
- Set parameters like friction coefficients and initial flow conditions. 

### Step 4: Run the Simulation
- Click the 'Run Simulation' button to start the process.
- Wait for the simulation to complete. The application will display results visually.

### Step 5: Analyze Results
- Use the built-in tools to analyze runout, velocity, and deposition heights.
- Save results for later review or publication if needed.
